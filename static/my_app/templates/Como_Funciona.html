O que é o PBL?

O PBL (Problem-Based Learning) é uma metodologia de ensino focada na resolução de problemas reais. Os alunos são desafiados a identificar, pesquisar e propor soluções para questões práticas, promovendo o desenvolvimento de habilidades como:

  • Pensamento crítico
  • Trabalho em equipe
  • Autonomia

Como Funciona o Questionário?

O questionário foi desenvolvido para avaliar as habilidades principais dos alunos durante o processo de PBL (Problem-Based Learning). Abaixo estão os principais elementos:

Estrutura do Questionário

O questionário será composto por 10 perguntas focadas nas habilidades desenvolvidas pelo aluno, como:

  • Resolução de problemas
  • Trabalho em equipe
  • Pensamento crítico

Escala de Avaliação

Os alunos responderão às perguntas utilizando uma escala, como:

  • Concordo totalmente
  • Concordo parcialmente
  • Discordo
  • Não sei

Coleta de Dados

As respostas serão armazenadas no banco de dados e associadas ao perfil de cada aluno, permitindo a geração de métricas personalizadas.
